# 🥖 Panadería & Repostería - Dulce Horno

> Plataforma de gestión y ventas para una panadería y repostería artesanal.

## 📋 Descripción

**Dulce Horno** es un sistema integral para la administración de una panadería y repostería. Permite gestionar productos, inventario, pedidos, y proveedores, además de ofrecer una interfaz amigable para que los clientes realicen pedidos en línea. Ideal para negocios que buscan digitalizar sus operaciones y mejorar la experiencia de compra.

## ✨ Características principales

- 🍞 Catálogo de productos (panes, pasteles, galletas, facturas, etc.)
- 🛒 Carrito de compras y sistema de pedidos en línea
- 📊 Dashboard de ventas y reportes diarios
- 📦 Control de inventario y materias primas (harina, huevos, levadura, etc.)
- 👥 Gestión de clientes y proveedores
- 🧾 Generación de tickets y facturas
- 🔐 Autenticación de usuarios (admin, vendedor, cliente)
- 📱 Diseño responsivo (funciona en móviles, tablets y escritorio)
